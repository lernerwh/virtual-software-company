# 需求描述

## 项目概述
开发一个Web网站，用于自动爬取和展示GitHub上star数量排名前500的Android开源项目信息。

## 核心功能需求
1. **数据爬取**
   - 自动获取GitHub上Android开源项目列表
   - 按star数量排序，获取Top 500项目
   - 提取项目信息：名称、描述、star数、fork数、地址、语言、更新时间等

2. **数据展示**
   - Web界面展示项目列表
   - 支持分页浏览
   - 支持搜索和筛选功能

3. **数据导出**
   - 一键导出为Excel格式
   - 支持自定义字段选择

## 技术需求
- 前端：响应式Web界面
- 后端：数据爬取和API服务
- 数据存储：项目数据缓存
- 导出功能：生成Excel文件

## 用户角色
- 普通用户：浏览项目列表、导出数据

## 原始需求
"帮我做一个web网站，里边可以自动爬取最新的github上的star数量top 500左右的android开源项目信息和地址，且能快速导出为excel"
