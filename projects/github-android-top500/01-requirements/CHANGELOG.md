### 新增
- 功能 8: 代码行数统计功能
  - 通过 git 浅拷贝 clone 项目到临时目录
  - 使用 cloc 工具统计各语言代码行数
  - 每个项目新增"统计代码"按钮
  - 实时展示统计进度
  - 统计结束后自动清理临时文件

### 技术风险
| 风险 | 影响 | 缓解措施 |
|------|------|----------|
| cloc 工具未安装 | 高 | 启动时检查，提示用户安装 |
