## 部署建议

### 前置条件
1. 安装 cloc 工具
   ```bash
   # macOS
   brew install cloc

   # Ubuntu/Debian
   sudo apt-get install cloc

   # Windows
   choco install cloc
   ```

2. 安装 git
   ```bash
   # 确保服务器已安装 git
   git --version
   ```

3. 数据库迁移
   ```bash
   # 应用会自动创建新表
   npm run dev
   ```

### 配置建议
- 设置合理的超时时间（建议 60-120 秒）
- 限制并发统计数量（当前为 1）
- 定期清理旧的统计任务记录
